# Student GradingSystem & History

A simple, console-based Java application that calculates student grade averages, evaluates academic honors, and maintains a session history log.

## 🚀 Features
* **Input Validation:** Prevents empty names, non-alphabetical characters, and non-numeric grade entries.
* **Automatic Evaluation:** Calculates the average, rounds the final grade, and assigns pass/fail remarks and honor standings.
* **Session History:** Keeps a formatted running log of all entered students during the active session.

## 📋 Academic Criteria
* **Passing Mark:** 75 and above is marked as Passed (below 75 is Failed).
* **Honors Thresholds:**
  * 96 and above: With Highest Honor
  * 94 to 95.99: With High Honor
  * 90 to 93.99: Honor
  * Below 90: No Honor
